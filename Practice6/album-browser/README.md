# Album Browser - Lab 6

Angular SPA that lets you browse albums and photos from the JSONPlaceholder API.

## How to run

```bash
cd lab6/album-browser
npm install
ng serve
```

Open http://localhost:4200

## Features
- View all albums
- View album details and edit the title
- View album photos in a grid
- Delete albums from the list

## API
Uses https://jsonplaceholder.typicode.com

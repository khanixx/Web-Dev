Album Browser

Album Browser is a full-featured Angular single-page application that demonstrates modern web development practices including component-based architecture, reactive programming with RxJS, and RESTful API integration.

The app connects to the JSONPlaceholder API to fetch, display, update, and delete album data — providing a real-world CRUD experience.

Features Implemented

  • ✅ Angular Router with 6 configured routes
  • ✅ Standalone components architecture
  • ✅ AlbumService with HttpClient for all API calls
  • ✅ TypeScript interfaces for Album and Photo models
  • ✅ Read albums list with navigation to detail view
  • ✅ Update album title with save functionality
  • ✅ Delete album with UI update
  • ✅ Photo grid view with thumbnails and hover titles
  • ✅ Loading indicators and empty state handling
  • ✅ Responsive CSS layout for all screen sizes
